## Getting Started

To get a local copy up and running follow these simple example steps.

### Installation

1. Clone the repo
   ```sh
   git clone git@github.com:Beefs4000/Testing.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run using intergrated terminal
   ```sh
   node app.js
   ```
